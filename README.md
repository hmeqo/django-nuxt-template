## Django-Nuxt-Template

> [!WARNING]
> 此模板处在实验性阶段

全局搜索 `Django Nuxt Template`, `DjangoNuxtTemplate` 与 `django-nuxt-template` 替换为你的项目名称

确认配置无误 `conf/crontab.conf` `conf/website.service`

### 用法

- 同步前后端运行环境

  ```bash
  ./manage.sh sync
  ```

- 开发

  ```bash
  ./manage.sh dev
  ```

- 部署
  
  ```bash
  ./manage.sh deploy
  ```
